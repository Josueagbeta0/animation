# Animation Turtle

Un petit script Python qui utilise le module `turtle` pour dessiner une animation colorée sur un fond noir.

## Description

Le script `animation.py` crée une animation de cercles en rotation avec des couleurs changeantes. Le tracé est réalisé à l'aide de la tortue graphique (`turtle`) et s'exécute dans une fenêtre interactive.

## Prérequis

- Python 3
- Le module standard `turtle` (inclus avec Python sur la plupart des installations)

## Utilisation

1. Ouvrez un terminal.
2. Placez-vous dans le dossier du projet.
3. Exécutez :

```bash
python animation.py
```

4. Une fenêtre Turtle s'ouvrira et affichera l'animation. Cliquez sur la fenêtre pour la fermer.

## Fichier

- `animation.py` : script principal qui génère l'animation.

## Notes

- La vitesse de dessin est réglée à la valeur maximale.
- Vous pouvez modifier la liste `col` dans `animation.py` pour tester d'autres palettes de couleurs.
